IMPORTANT

Please make a note of below point

  • Use Google Chrome Browser version 53.0 or above, as this tool has been tested for this plateform only. Some functionaliy might break or/and appearence might change on other browser
  • If you suspect, some processing issue on backend server, then use WEBCONSOLE feature to analyse the same, which might help you fix the issue from your end if any
  • If you suspect, some processing issue at browser side, then use GOOGLE browser console to see the log, which might help you fixing the issue from your end if any.
  • If you suspect some issue after importing any data, then try clearing browser cache then enter main http url then enter those data manually. Such issue might be due to unexpected imported data
  • If are you not using WEBCONSOLE then, make enable_log false in CONFIGURATION TAB, to save little backend load
ABOUT

This is implementation of iso8583 standard. we can use this tool to generate iso8583 message, send and receive to/from any server that understand iso8583. End user of this tool will use standard web interface provide by browser. This tool has been designed and developed from end user percpective. we will see technical and function aspect in subsequent chapter.Below are some striking point of this tool.

  • Full of feature
  • simple and clean UI
  • Highly configurable
  • Multiuser
  • Easy plugin of future feature request
  • No deployable to enduser, just http url
  • deploy backend at centralized server
